### 산술 연산자

#### 개요 및 종류

사칙연산(+, -, \*, /)과 나누기 연산(%)

### 비트 연산자

#### 개요

데이터를 비트 단위로 연산

#### 종류

- 비트 이동 연산자 (`>>` `<<` `>>>`)
- 비트 논리 연산자 (`&` `|` `^` `~`)

### 관계 연산자

#### 개요

크고 작음과 같고 다름을 비교

#### 종류

- `>` `<` `>=` `<=` `==` `!=`

### 논리 연산자

#### 개요

'그리고(AND)'와 '또는(OR)'으로 조건을 연결

#### 종류

- `&&` `||` `!` `&` `|` `^` `~`
